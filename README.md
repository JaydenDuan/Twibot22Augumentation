# Twibot22Augumentation

Twibot22Augumentation — sample data for diffusion-based augmentation on TwiBot-22.

This repository hosts a **small sample** of the data used in our diffusion-based debiasing study on TwiBot-22 for LLM-driven social bot detection.

> **Note**: The original TwiBot-22 dataset is licensed and cannot be redistributed here. We only release derived artifacts (synthetic feature vectors, prompt-format JSONL samples, and evaluation metrics) that do not contain raw TwiBot user records.

## Repository Layout

```
.
├── finetune_data_sample/      # Prompt-format JSONL samples (200 train / 50 val)
│   ├── train_diffusion_v2_sample.jsonl
│   ├── train_std_v2_sample.jsonl
│   ├── val_diffusion_v2_sample.jsonl
│   └── val_std_v2_sample.jsonl
├── diffusion_model_v2/        # Tabular DDPM artifacts
│   ├── meta.json              # Model configuration metadata
│   └── synthetic_samples.json # Generated synthetic low-activity human samples
└── results/
    └── summary.json           # Per-model FPR/ACC summary on TwiBot-22
```

## Data Description

### `finetune_data_sample/`

JSONL files in OpenAI fine-tuning format. Each line is a triple of `system` / `user` / `assistant` messages.

| File | Lines | Description |
|------|-------|-------------|
| `train_diffusion_v2_sample.jsonl` | 200 | Sample of training data **with** diffusion-based augmentation (synthetic low-activity human samples mixed in) |
| `train_std_v2_sample.jsonl` | 200 | Sample of standard training data (no augmentation) |
| `val_diffusion_v2_sample.jsonl` | 50 | Validation sample (diffusion-augmented) |
| `val_std_v2_sample.jsonl` | 50 | Validation sample (standard) |

The full training set contains 7,457 (diff) / 6,557 (std) lines.

### `diffusion_model_v2/`

Artifacts from training a Tabular DDPM on real low-activity human users.

- `meta.json` — Hyperparameters and configuration of the diffusion model.
- `synthetic_samples.json` — 1,000 synthetic feature vectors generated by the model. Each entry contains a numerical feature record for an artificial low-activity human user.

### `results/`

- `summary.json` — Aggregate evaluation metrics (ACC, FPR, per-group FPR) on the TwiBot-22 test set across four LLM backbones (GPT-4o-mini, Llama-3-8B, Mixtral-8×7B, Qwen3.5-9B), for both standard and diffusion-augmented fine-tuning.
